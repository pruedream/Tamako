# 部署的相关配置



后端

nohup /usr/local/java/java17/bin/java \
-Xms512m -Xmx1024m \
-Dshort-link.domain.default=112.74.36.216:8003 \
-jar /home/shortlink/shortlink-aggregation.jar > logs/shortlink-aggregation.file 2>&1 &
~~~
